## 시작하기

### 계정 생성

1. [가입 페이지](https://example.com/signup)로 이동합니다
2. 이메일 주소와 비밀번호를 입력합니다
3. "가입하기" 버튼을 클릭합니다
4. 이메일로 전송된 인증 링크를 클릭합니다

![회원가입](images/signup.png)

### 로그인

1. [로그인 페이지](https://example.com/login)로 이동합니다
2. 이메일과 비밀번호를 입력합니다
3. "로그인" 버튼을 클릭합니다

## 주요 기능

### 기능 1: 사용자 관리

사용자 정보를 관리할 수 있습니다.

**가능한 작업:**
- ✅ 프로필 정보 수정
- ✅ 비밀번호 변경
- ✅ 프로필 이미지 업로드
- ✅ 계정 삭제

**사용 방법:**
1. 우측 상단의 프로필 아이콘을 클릭합니다
2. "설정"을 선택합니다
3. 원하는 항목을 수정합니다
4. "저장" 버튼을 클릭합니다

![프로필 설정](images/profile-settings.png)

### 기능 2: 데이터 관리

데이터를 생성, 조회, 수정, 삭제할 수 있습니다.

## 단계별 가이드

### 가이드 1: 첫 프로젝트 생성하기

#### Step 1: 프로젝트 생성

1. 대시보드에서 "+ 새 프로젝트" 버튼을 클릭합니다
2. 프로젝트 이름을 입력합니다
3. 템플릿을 선택합니다 (선택사항)
4. "생성" 버튼을 클릭합니다

![프로젝트 생성](images/create-project.png)

#### Step 2: 설정 구성

1. 프로젝트 설정 페이지로 이동합니다
2. 필요한 옵션을 선택합니다
3. "저장" 버튼을 클릭합니다

#### Step 3: 팀원 초대

1. "팀" 탭으로 이동합니다
2. "+ 팀원 초대" 버튼을 클릭합니다
3. 이메일 주소를 입력합니다
4. 권한을 선택합니다 (관리자/멤버)
5. "초대" 버튼을 클릭합니다

## FAQ

### Q: 비밀번호를 잊어버렸어요
A: 로그인 페이지에서 "비밀번호 찾기"를 클릭하고 이메일 주소를 입력하세요. 비밀번호 재설정 링크가 전송됩니다.

### Q: 이메일 인증이 오지 않아요
A: 스팸 폴더를 확인하거나, "인증 메일 재전송" 버튼을 클릭하세요.

### Q: 계정을 삭제하고 싶어요
A: 설정 > 계정 > "계정 삭제"를 클릭하세요. 삭제된 계정은 복구할 수 없습니다.

---

## 트러블슈팅

### 로그인이 안 돼요

**증상:** 로그인 버튼을 클릭해도 반응이 없습니다

**해결 방법:**
1. 브라우저 쿠키를 삭제합니다
2. 브라우저 캐시를 지웁니다
3. 다른 브라우저로 시도합니다
4. 문제가 계속되면 support@example.com으로 문의하세요

### 파일 업로드가 실패해요

**증상:** 파일 업로드 시 에러 메시지가 나타납니다

**원인:**
- 파일 크기가 5MB를 초과함
- 지원하지 않는 파일 형식

**해결 방법:**
1. 파일 크기를 5MB 이하로 줄입니다
2. 지원되는 형식(JPG, PNG, PDF)으로 변환합니다
